Item Text

Description of the IRW item text data

The below scatterplot shows the mean number of words (when item_text is split by ’ ’) and characters in the items for the IRW tables with associated item text (along with the name of the relevant psychometric instrument).

Code 
Plot = require("@observablehq/plot")

// Transpose the data for Observable
plotData = transpose(plot_data)

// Create scatterplot with enhanced tooltips
Plot.plot({
  marks: [
    Plot.dot(plotData, {
      x: "mean_word",
      y: "mean_character",
      title: d => `Table: ${d.table}\nInstrument: ${d.instrument}\nMean Word: ${d.mean_word}\nMean Characters: ${d.mean_character}`,
      tip: true
    })
  ],
  x: {
    label: "Mean Word"
  },
  y: {
    label: "Mean Characters"
  }
})
